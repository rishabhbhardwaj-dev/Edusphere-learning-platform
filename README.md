# 🎓 EduSphere Platform

A full-stack ed-tech platform built using the MERN stack and FastAPI, designed to support interactive online learning with authentication, course management, and AI-powered features.

---

## 🚀 Features

- 👨‍🏫 Instructor & Student modules
- 🔐 JWT Authentication
- 📧 OTP-based email verification
- 📚 Course creation and enrollment
- 📈 Student progress tracking
- 💳 Razorpay payment integration
- ☁️ Cloudinary media management
- 🤖 AI-powered microservices
- 🛡️ Role-based access control

---

## 🛠️ Tech Stack

### Frontend

### Database & Services
- MongoDB Atlas

---

## 📌 Project Overview

EduSphere is a scalable online learning platform that enables instructors to create and manage courses while allowing students to enroll, learn, and track their progress efficiently.

The platform integrates modern authentication systems, AI microservices, and payment solutions to deliver a complete ed-tech ecosystem.
